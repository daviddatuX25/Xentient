Node Lifecycle

A node moves gracefully through defined states. Click below to explore what happens during outages or resets.

📦 UNCLAIMED
🔌 BARE
🟢 OPERATIONAL
⚠️ DEGRADED

OPERATIONAL

Peripherals enumerated. Profile active. Data mode running. Normal runtime state. Streams data upstream.

Transition: Upstream loss → DEGRADED | Peripherals removed → BARE

Data Contract

Every packet is self-describing. Upstream routing is handled seamlessly by the Harness based on the payload fields.

Field Type Purpose
space_id string Logical space context
node_id string Physical emitter identifier
peripheral_type enum Sense/actuator category
timestamp_utc ISO 8601 NTP-synced capture time
sequence uint32 Monotonic counter for ordering
mode enum push / stream / fetch / adaptive
payload bytes Audio chunk, JPEG, sensor event
