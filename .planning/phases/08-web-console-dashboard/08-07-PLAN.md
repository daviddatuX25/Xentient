# 08-07: ControlServer Refactoring

## Goal

Replace the hand-rolled `if/else` routing in `ControlServer.handleRequest()` with a zero-dependency `MicroRouter` class that supports method matching, path parameters (`:id`), and structured error responses. Expand the constructor dependency injection to include `SpaceManager`, `EventBridge`, `PackLoader`, and `SkillLog` (same pattern as `McpToolDeps`).

## Why First (Wave 1 Blocker)

All new REST endpoints (08-01) and SSE events (08-02) depend on:
1. A route table that can handle ~20 routes cleanly
2. ControlServer having access to SpaceManager, PackLoader, EventBridge, SkillLog

Without this refactoring, adding 15 new endpoints to the existing `if/else` chain becomes unmaintainable.

## Implementation

### Step 1: Create `MicroRouter` class

**File:** `harness/src/comms/MicroRouter.ts` (new, ~70 lines)

```typescript
type RouteHandler = (req: IncomingMessage, res: ServerResponse, params: Record<string, string>) => Promise<void>;

interface Route {
  method: string;
  pattern: RegExp;
  paramNames: string[];
  handler: RouteHandler;
}

export class MicroRouter {
  private routes: Route[] = [];

  add(method: string, path: string, handler: RouteHandler): this {
    const paramNames: string[] = [];
    const patternStr = path.replace(/:(\w+)/g, (_m, name) => {
      paramNames.push(name);
      return `(?<${name}>[^/]+)`;
    });
    const pattern = new RegExp(`^${patternStr}/?$`);
    this.routes.push({ method: method.toUpperCase(), pattern, paramNames, handler });
    return this;
  }

  resolve(method: string, url: string): { handler: RouteHandler; params: Record<string, string> } | { status: number } {
    const pathname = url.split('?')[0];
    const methodUpper = method.toUpperCase();
    let pathMatched = false;

    for (const route of this.routes) {
      const match = route.pattern.exec(pathname);
      if (!match) continue;
      pathMatched = true;
      if (route.method !== methodUpper) continue;

      const params: Record<string, string> = {};
      if (match.groups) {
        for (const name of route.paramNames) {
          if (match.groups[name] !== undefined) params[name] = match.groups[name];
        }
      }
      return { handler: route.handler, params };
    }
    return { status: pathMatched ? 405 : 404 };
  }
}
```

Key design decisions:
- Named capture groups (`(?<id>[^/]+)`) for parameter extraction
- 404 vs 405 distinction (path matched but method didn't → 405 Method Not Allowed)
- Fluent API (`router.add().add().add()`)
- Zero external dependencies

### Step 3: Refactor `handleRequest()` to use `MicroRouter`

Replace the existing `if/else` chain in `ControlServer.handleRequest()`:

```typescript
// Before: if (url === "/api/mode" && method === "POST") { ... }
// After:
private router = new MicroRouter();

constructor(private deps: ControlServerDeps, port: number) {
  super();
  this.registerRoutes();
  // ... existing SSE, static file setup ...
}

private registerRoutes(): void {
  this.router
    .add('GET',  '/api/status',        this.handleGetStatus)
    .add('GET',  '/api/sensors',       this.handleGetSensors)
    .add('GET',  '/api/mode',          this.handleGetMode)
    .add('POST', '/api/mode',          this.handleSetMode)
    .add('POST', '/api/trigger',       this.handleTrigger)
    .add('POST', '/api/vad',           this.handleVad)
    .add('POST', '/api/text',          this.handleText)
    .add('GET',  '/api/camera',        this.handleGetCamera);
    // New endpoints added in 08-01
}

private async handleRequest(req: IncomingMessage, res: ServerResponse): Promise<void> {
  const method = req.method ?? 'GET';
  const url = req.url ?? '/';

  // CORS
  this.setCORSHeaders(res);
  if (method === 'OPTIONS') { res.writeHead(204); res.end(); return; }

  // SSE (long-lived, skip router)
  if (url.split('?')[0] === '/api/events') { this.handleSSE(req, res); return; }

  // Route resolution
  const result = this.router.resolve(method, url);
  if ('status' in result) {
    const msgs: Record<number, string> = { 404: 'Not found', 405: 'Method not allowed' };
    this.sendJSON(res, result.status, { error: msgs[result.status] ?? 'Bad request' });
    return;
  }

  try {
    await result.handler(req, res, result.params);
  } catch (err) {
    this.sendJSON(res, 500, { error: 'Internal server error' });
  }
}
```

private async parseBody(req: IncomingMessage): Promise<unknown> {
  return new Promise((resolve, reject) => {
    const chunks: Buffer[] = [];
    req.on('data', (c) => chunks.push(c));
    req.on('end', () => {
      const body = Buffer.concat(chunks).toString();
      if (!body) { resolve({}); return; }
      try { resolve(JSON.parse(body)); }
      catch (e) { reject(new Error('Invalid JSON')); }
    });
    req.on('error', reject);
  });
}

## Edge Cases

- **Trailing slashes:** `/api/skills/` and `/api/skills` must both resolve to the same route. The regex pattern `^/api/skills/?$` handles this.
- **Query strings:** `url.split('?')[0]` strips query params before routing.
- **CORS preflight:** `OPTIONS` requests must bypass the router and return 204 immediately.
- **SSE endpoint:** Must bypass the router entirely (long-lived connection).
- **Static files:** If no API route matches, fall through to static file serving (existing behavior).
